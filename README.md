## 如何运行？

> 安装依赖

在项目根目录执行

```bash
npm install
//或
yarn install
```

> Mac电脑运行

安装pod依赖

```bash
npx pod-install
```

然后运行 

```bash
npm run ios
```

>Windows电脑运行

```bash
npm run android
```
